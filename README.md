# Elanur Oğuz | Portfolyo Web Sitesi

Bu proje, **Elanur Oğuz (Bilgisayar Programcısı)** için hazırlanmış modern, hızlı ve duyarlı (responsive) bir portfolyo web sitesidir. 

Next.js App Router mimarisi ve saf CSS kullanılarak, modern ve akıcı animasyonlarla zenginleştirilmiş premium Gece Mavisi karanlık tema tonlarında tasarlanmıştır.

## 🚀 Canlı Web Sitesi
Sitenin internetteki canlı haline aşağıdaki adresten ulaşabilirsiniz:
👉 **[https://elanuroguz-portfolyo.vercel.app](https://elanuroguz-portfolyo.vercel.app)**

*(Not: Vercel üzerinde oluşturulan alternatif adres: [https://elanuroguz-portfolyo-elanuroguzzs-projects.vercel.app](https://elanuroguz-portfolyo-elanuroguzzs-projects.vercel.app))*

---

## 🛠️ Kullanılan Teknolojiler
- **Framework:** Next.js 15+ (App Router)
- **Kütüphane:** React 19
- **Stil:** Vanilla CSS (CSS Variables, Glassmorphism, Custom Animations & Keyframes)
- **Tasarım Konsepti:** Premium Dark Mode (Gece Mavisi) ve Responsive Mobil Uyumlu Arayüz

---

## 📁 Proje Sayfaları
1. **Ana Sayfa:** Karşılama başlığı, unvan, dinamik morphing animasyonlu görsel ve hızlı yönlendirme butonları.
2. **Hakkımda:** Eğitim hedefleri, vizyon ve gelecek hedefleri (OOP, Veritabanı Yönetimi, Agile/Scrum).
3. **Eğitim & Yetenekler:** Programlama dilleri (Python, C#), Veri tabanı (SQL, DBMS), Araçlar (Git, GitHub, VS Code, Visual Studio), Eğitim kronolojik zaman çizelgesi (Timeline) ve sertifikalar.
4. **İletişim:** E-posta, LinkedIn, GitHub iletişim kanalları ve etkileşimli mesaj formu.

---

## 💻 Yerel Geliştirme (Local Development)

Projeyi kendi bilgisayarınızda çalıştırmak için:

1. Bağımlılıkları yükleyin:
```bash
npm install
```

2. Geliştirme sunucusunu başlatın:
```bash
npm run dev
```

Tarayıcınızda `http://localhost:3000` adresini açarak siteyi görüntüleyebilirsiniz.
